# Library Management System

A simple library management system built with Python using OOP principles.

## Features

- Add books
- Register users
- Borrow books
- Return books
- Track loans

## Classes

- Library
- Book
- Author
- User
- Employee
- Loan

## OOP Concepts

- Encapsulation
- Abstraction
- Polymorphism
- Inheritance (optional)

## Run

```bash
python main.py
